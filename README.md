# ClearCredits
A service-backed credit infrastructure for issuing, tracking, and redeeming prepaid value through a transparent ledger.

## Repository Structure

- `docs/`
  - `vision.md` — why Clear Credits exists and the principles it follows
  - `ledger-model.md` — how credits are issued, tracked, and protected
- `examples/`
  - `sample-ledger-events.json` — example ledger events showing issuance and redemption
